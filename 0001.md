
# 生成200个激活码
```python
import random,string

forselect=string.ascii_letters+string.digits

def generate(count,length):
    for x in range(count):
        re=""
        for j in range(length):
            re+=random.choice(forselect)
        print(re)

if __name__=="__main__":
    generate(200,20)
```
